ЦЕЛЬ ПРОЕКТА
------------
Одностраничное приложение (один HTML) для глубокого фокуса, заметок, идей и
продуктивности. Всё хранится локально в браузере. Приватность — ключевая ценность.

ФАЙЛЫ ДЛЯ ПЕРЕДАЧИ ЛЮБОМУ AI-АГЕНТУ (читать все три)
----------------------------------------------------
1. статус-проекта.txt      — ЭТОТ ФАЙЛ (сделано, архитектура, правила)
2. план.txt                — roadmap и этапы (вкл. Этап 5 — фото)
3. сложные-задачи-позже.txt — отложенные сложные фичи (4.2+)
Дополнительно: обзор-проекта.txt — приоритеты и дорожная карта

ЖЁСТКИЕ ПРАВИЛА ДЛЯ AI-АГЕНТА
-----------------------------
1. НЕ удалять и НЕ переписывать большие куски работающего кода без необходимости
2. Только точечные изменения в index.html
3. Не трогать Этапы, которые пользователь не просил
4. После изменений — кратко: что изменено / что не тронуто / что проверить
5. Шифрование требует HTTPS или localhost (не file://)
6. При сомнениях — СПРАШИВАТЬ пользователя, не импровизировать
7. Удаление всех данных — ТОЛЬКО Secure Wipe (не возвращать «очистить всё»)

АРХИТЕКТУРА ДАННЫХ (НЕ ЛОМАТЬ)
================================================================================

БЕЗ ЗАЩИТЫ — ключи localStorage:
  flownote_saved_notes, flownote_saved_ideas, flownote_custom_prompts
  flownote_focus_sessions, flownote_current_draft
  flownote_theme, flownote_welcomed, flownote_shortcut_os

С ЗАЩИТОЙ — один blob: flownote_security_v2

Все чтения/записи — через getAppData() / setAppData() / removeAppData()

Инкогнито блокирует setAppData для пользовательских ключей (INCOGNITO_BLOCKED_KEYS).

Структура заметки:
  { id, content, created, modified, wordCount, destruct?, destructAt? }

Структура идеи:
  { id, text, category, created, destruct?, destructAt? }

Self-destruct: getDestructSettings(), applyDestructToItem(), processAllSelfDestruct()
Watcher: startSelfDestructWatcher() — каждые 30 сек

Глубокий фокус: isDeepFocusMode, toggleDeepFocus(), syncDeepFocusTimer()
