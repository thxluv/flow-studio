Условия использования

Дата вступления в силу: 8 июля 2026

Важно: Используя Сервисы, вы принимаете настоящие Условия и Политику конфиденциальности. Сервисы — «как есть», без SLA.

1. Общие положения

1.1. FlowNote, FlowPhoto и FlowVault предоставляются физическим лицом как экспериментальный pet-проект.

1.2. Сервисы не являются ОРИ, соцсетью или публичной площадкой; общедоступного каталога контента нет.

1.3. Возраст Пользователя — не менее 14 лет. Лица 14–18 лет — под ответственностью законных представителей.

2. Сервисы

FlowNote — заметки локально в браузере; опциональное шифрование на устройстве. Экспорт JSON — на риск Пользователя.

FlowPhoto — фото шифруются в браузере; EXIF удаляется до отправки; имя файла на сервере анонимизируется; доступ по приватной ссылке с ключом в #.

FlowVault — личный сейф фото; пароль сейфа — хеш на сервере; ключи фото — у Пользователя.

3. Ответственность Пользователя

3.1. Пользователь несёт полную ответственность за свой Контент и последствия передачи приватных ссылок.

3.2. Запрещён незаконный контент (в т.ч. CSAM, экстремизм, вредоносное ПО, спам, атаки на инфраструктуру). Загрузка Ciphertext не означает одобрения со стороны Оператора.

4. Ограничение ответственности Оператора

4.1. Оператор не отвечает за утрату данных, сбои хостинга, действия третьих лиц с ссылками/ключами, косвенные убытки.

4.2. При бесплатном использовании совокупная ответственность Оператора — 0 ₽. FlowPhoto и FlowVault предоставляются бесплатно.

5. Удаление и обращения третьих лиц

5.1. Пользователь удаляет данные через интерфейс (localStorage, срок хранения, FlowVault, уничтожение ссылок).

5.2. Удаление Ciphertext по требованию третьих лиц (жалобы, претензии) — только по решению суда или обязательному требованию уполномоченного госоргана РФ, в объёме документа (по ID / URL). Иные обращения Оператор вправе не исполнять. Подробнее — для уполномоченных органов.

6. FlowNote Pro

Платные функции могут предоставляться только в FlowNote (подписка или разовая оплата). FlowPhoto в Pro не входит. Условия оплаты уточняются при подключении; до этого Pro может быть недоступен.

7. Интеллектуальная собственность

Права на код Сервисов принадлежат Оператору. Копирование и использование без разрешения запрещено. Контент Пользователя остаётся его собственностью; для работы Сервиса Пользователь предоставляет право хранить и передавать Ciphertext.
