--------------------------------------------------------------------------------
ЭТАП 5 — ЗАШИФРОВАННОЕ ФОТО-ХРАНИЛИЩЕ + ПЕРЕДАЧА
--------------------------------------------------------------------------------

5.1 — Локальный «Сейф» для изображений
  • Новая секция в приложении (или вкладка «Сейф»)
  • Хранение в IndexedDB (localStorage не подходит для фото)
  • Превью, название, дата, удаление одного файла
  • Если включена защита — шифрование через существующий vault / Web Crypto
  • Лимит размера на файл (например 10 МБ) — предупреждение пользователю
  Риск: средний | Не ломает текущий слой getAppData/setAppData для текста

5.2 — Зашифрованная передача файла (.fnshare)
  • Кнопка «Поделиться» у фото → пароль → скачать .fnshare
  • Получатель: Импорт .fnshare в FlowNote → ввод пароля → просмотр
  • Формат: JSON { version, iv, ciphertext, meta: { name, mime, created } }
  • Zero-knowledge: пароль не хранится, только в голове отправителя/получателя
  Риск: средний

5.3 — Портативная HTML-«ссылка» (опционально)
  • Генерация одного .html с зашифрованным blob внутри + мини UI расшифровки
  • Можно залить на GitHub Pages / отдать файлом — работает без FlowNote
  Риск: средний | Размер файла = размер фото + overhead

5.4 — Короткая URL-ссылка (только при сервере, далеко)
  • Загрузка зашифрованного blob на хостинг
  • Короткая ссылка + пароль отдельно
  • Противоречит «всё локально» — отдельный проект

Рекомендуемый порядок: 5.1 → 5.2 → 5.3. Без сервера.

1. Удобство и интуитивная понятность (UX-фичи)
Командная палитра (Command Palette)
Как в VS Code или Notion. Жмёшь Ctrl/Cmd + K — появляется поиск по всем инструментам, быстрые действия («Новая заметка», «Сгенерировать идею», «Экспорт всего», «Заблокировать»).
Очень сильно ускоряет работу и делает приложение «взрослым».
Глобальный поиск + теги
Один поиск по всему приложению (заметки + идеи + промпты + сессии).
Плюс система тегов (#продуктивность, #идея_для_бизнеса и т.д.). Можно фильтровать по тегам везде.
Drag & Drop + организация

Перетаскивать заметки в папки/проекты
Drag заметки прямо в генератор идей или в промпты
Визуальные доски (Kanban-стиль) для идей и задач

Фокус-режимы

«Глубокий фокус» — скрывает всё кроме блокнота + большой таймер + минималистичный интерфейс
«Только идеи» — режим для мозгового штурма
Полноэкранный режим с горячей клавишей

Шаблоны и быстрый старт
Кнопка «Новый →» с готовыми шаблонами:

Утренний разбор дня
Еженедельная рефлексия
План проекта
Промпт для конкретной задачи
Можно создавать свои шаблоны.

Undo / Redo + история изменений
Полноценная история изменений внутри заметки (как в Google Docs). Можно откатить даже после перезагрузки страницы.
Визуальная статистика
Дашборд не просто цифры, а красивые графики:

Сколько времени в фокусе по дням/неделям
Самые продуктивные часы
Рост количества идей
(Можно на Chart.js или ApexCharts — всё ещё один файл)

PWA + оффлайн + мобильная версия
Сделать нормальное PWA (устанавливается как приложение). Работает даже без интернета. На телефоне будет ощущаться как нативное приложение.
Горячие клавиши везде

Ctrl + S — сохранить
Ctrl + Shift + I — новая идея
Ctrl + Shift + P — открыть промпты
Esc — выйти из фокус-режима
И т.д.

Быстрый захват (Quick Capture)
Отдельная маленькая кнопка/горячая клавиша, которая открывает мини-окно «Быстро записать мысль» — и она сразу попадает в Inbox / FlowNote.
2. Защита данных и приватность (darknet-level идеи)
Вот здесь самое интересное. Делаем так, чтобы даже если кто-то получит доступ к файлу/браузеру — данные были в безопасности. Идеи по возрастанию «жёсткости».
Базовый уровень (рекомендую начать с этого)

Мастер-пароль + шифрование localStorage (Web Crypto API + Argon2)
При открытии приложения просит пароль. Всё, что лежит в localStorage, зашифровано. Без пароля — просто пустой интерфейс.
Автоблокировка после 5–15 минут бездействия
Зашифрованный экспорт (при экспорте в JSON/PDF можно ставить пароль)

Средний уровень (уже очень серьёзно)

Понятная денеабельность (Plausible Deniability)
Как в VeraCrypt. Можно создать «ложный» пароль, который открывает чистую версию приложения (без важных заметок). Реальные данные открываются только по другому паролю.
Несколько «сводов» (Vaults)
Основной + «Секретный» + «Для семьи» и т.д. Каждый со своим паролем. Можно быстро переключаться.
Self-destruct заметки
Заметка с таймером самоуничтожения (через 1 час / 1 день / после прочтения). После этого она исчезает навсегда.
Зашифрованная передача
Генерировать зашифрованную ссылку/текст, который можно отправить другому человеку (он расшифрует только со своим паролем).

Жёсткий уровень (darknet-стиль)

Zero-knowledge по умолчанию — приложение вообще ничего не знает о тебе. Даже если мы потом добавим синхронизацию — только зашифрованные данные.
Steganography (опционально)
Возможность прятать важные заметки внутри обычных картинок (типа фото котика).
Tor-friendly режим
Отключать все внешние CDN и скрипты, оставлять только локальный код (чтобы приложение комфортно работало через Tor Browser).
Desktop-версия через Tauri (следующий большой шаг)
Упаковать приложение в настоящую десктопную программу. Тогда можно будет использовать настоящие системные возможности защиты (ключницы ОС, биометрия, изоляция процессов). Это уже совсем другой уровень.
Локальный аудит-лог
Журнал всех действий внутри приложения (кто и когда открывал/менял заметки). Можно зашифровать отдельно.
Secure Wipe
Функция «Полностью уничтожить все данные» с перезаписью (чтобы даже при forensic-анализе было сложно восстановить).

Дополнительные фишки приватности

Режим «Incognito внутри приложения» — временная сессия, которая ничего не сохраняет.
Возможность полностью отключить все внешние запросы (чтобы приложение работало в полностью изолированной среде).
Шифрование при экспорте в PDF (защищённый PDF-паролем).
